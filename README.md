Design Reasoning

Our game takes inspiration from classic 8-bit and 16-bit platformers. Growing up, we played Super Mario Bros, Mega Man Games, and Contra, and they had a massive impact on our game development. Older games often focused on tight controls and clear goals, and this project follows that by emphasizing speed, navigation, and awareness of the environment. Instead of enemies that deal damage, the large monsters in the game act as obstacles that slow the player down, keeping the game challenging. This supports a more momentum-based gameplay style while still fitting the feel of retro games. The different levels resemble the structured designs of older games. The main objective is to escape the game.  This reflects a classic motivation found in many retro games: keep moving, avoid setbacks, and reach the end. The game combines nostalgic visuals with smoother physics to create a modern take on a familiar style.

Levels were designed to gradually increase difficulty through layout complexity rather than enemy aggression, emphasizing speed and game awareness.
